## 角色設計

### 外觀
- **類型**：人類騎士
- **未來擴展**：可能支援角色自定義（依素材而定）

### 能力（第一階段）

| 能力 | 說明 |
|------|------|
| **移動** | 基本八方向移動 |
| **跳躍** | 基本跳躍 |
| **二段跳** | 空中可再跳一次，任何時候都能使用 |
| **衝刺** | 按住加速跑，地面/空中皆可使用，無限使用無冷卻 |

### 能力（第二階段）
- 武器攻擊
- 技能系統
- 等級成長

## 關卡設計

### 結構
- **類型**：線性關卡（起點→終點）
- **流程**：連續闘關
- **初期數量**：1-2 關
- **初期風格**：方塊原型關卡（Prototype）

### 檢查點系統
- **觸發方式**：自動觸發（進入區域即儲存）
- **重生機制**：死亡後從最近檢查點重新開始，無限重試

### 關卡主題（未來）
- 依可用素材決定
- 可能方向：草原、城市、工廠等

---

## 生存機制

### 血量系統
- 角色擁有血量值
- 血量歸零時死亡，從檢查點重生

### 傷害來源

| 來源 | 效果 |
|------|------|
| **陷阱** | 扣除血量 |
| **墜崖** | 扣除部分血量，重生於墜落前的安全位置 |

---

## 收集系統

### 收集品
- **數量**：每關 5 個
- **類型**：統一物品（如星星、寶石等）
- **分布**：主路線 + 隱藏區域

### 回饋
- **第一階段**：純成就收藏
- **未來可能**：解鎖獎勵（角色外觀、額外關卡等）

## 輸入配置

### 鍵盤滑鼠

| 操作 | 按鍵 |
|------|------|
| 移動 | WASD |
| 跳躍 | Space |
| 衝刺 | Shift（按住） |
| 視角旋轉 | 滑鼠移動 |
| 暫停 | ESC |

### 控制器

| 操作 | 按鍵 |
|------|------|
| 移動 | 左搖桿 |
| 跳躍 | A / X（南按鈕） |
| 衝刺 | RT / R2（按住） |
| 視角旋轉 | 右搖桿 |
| 暫停 | Start / Options |

### 虛擬搖桿（觸控）

| 位置 | 功能 |
|------|------|
| **左側** | 虛擬搖桿（移動） |
| **右側** | 跳躍按鈕 + 衝刺按鈕 |
| **右上角** | 暫停按鈕 |

### 暫停選單

| 選項 | 功能 |
|------|------|
| 繼續 | 關閉選單，繼續遊戲 |
| 重新開始 | 從關卡起點重新開始 |
| 設定 | 音量、靈敏度等設定 |
| 退出 | 返回主選單 |

### 主選單（基本）
- 開始遊戲
- 關卡選擇（顯示收集進度）
- 設定
- 退出
